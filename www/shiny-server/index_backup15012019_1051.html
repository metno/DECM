Beta version of DECM application for visualising and exploring climate model data.

To evaluate climate models on a global and regional scales, we propose a straightforward comparison of climate models' ability to reproduce statistical properties such as the seasonal cycle, based on large multi-model ensembles climate model simulations and the predefined IPCC regions. The displayed figure shows an example with simulated (colored lines) and observed-based (dashed black line) seasonal cycle for the 'Global' domain (default) and all simulations.

You can open each window and modify parameters that fit your needs - just have a look at our demo videos. Finally, you can use the app within this website or open a full-screen version.

Summary table of available DECM web applications

The various applications are based on CMIP5 and EURO-CORDEX simulations for climate variables such as precitpiation and temperature assuming the intermediate (RCP4.5) and the high (RCP8.5) emission scenarios. These are listed in the following table.

App Short description Acess the web app
DECMapp Contains all-in-one application for Data Evaluation of Climate Models Try me
GCMeval Specific app for climate model ensemble evaluation based on ensemble selection and model ranking Try me
Taylor Diagram app Specific app for evaluating climate model simulations using taylor diagrams Try me
Climate Model explorer Specific web app which contains tools for exploring meta data of CMIP5 and EURO-CORDEX simulations for climate variables such as precitpiation and temperature assuming both RCP4.5 and RCP8.5 Try me

Read more about the DECM applications

  • Sort through data available from the KNMI climate data explorer and produce simple plots and maps of climate change: Climate Model Explorer.
  • Evaluate the quality of RCM data by comparison with a reference data set, displayed in a Taylor diagram showing spatial correlation and standard deviation: RCM evaluation app.
  • The P vs T app shows the regionally averaged projected changes in precipitation and temperature for CMIP5 models. This tool can be used for selecting a smaller but representative ensemble of GCMs out of a large ensemble. The chosen models should cover the full range of projected climate change of all the available models, e.g., one from each quadrant of change (warm-wet, warm-dry, cold-wet, cold-dry) and one from the middle.
    • The seasonal cycle app shows the seasonal cycle of precipitation and temperature for GCMs (CMIP5) compared to a reference data set (ERAinterim). This tool can be used in ensemble selection to exclude models that are not representing the seasonal cycle well in a particular region.
  • An application for evaluation of CMIP5 climate model simulations with several performance statistics and visual representations of the model ensemble.
  • DECM full applications combines all the different applications into one using a categorised dashboard intended for product users, data users, and sectoral users.
